JavaScript

JavaScript ("JS" for short) is a cross-platform dynamic scripting language that can add interactivity to a website. JS is case-sensitive and uses the Unicode character set.

Comments

Comments can be one line or multi-line.

// one line comment

/* multi-line
comment
*/
Declarations
    There are 3 types of variable declarations in JS.
  • var - function-scoped or globally-scoped
  • let - block-scoped
  • const-block-scoped, should be initialized with a value.
Data types

The latest ECMAScript standard defines 7 Primitive data types- Boolean, null, undefined, Number, BigInt, String, Symbol and Object

Operators
    JS has the following types of operators with precedence listed in ascending order.
  1. Comma
  2. Assignment operators
  3. Conditional (ternary) operator
  4. Logical OR (||)
  5. Logical AND (&&)
  6. Bitwise OR (|)
  7. Bitwise XOR(^)
  8. Bitwise AND (&)
  9. Equality(==), Inequality(!=), Strict Equality(===), Strict Inequality(!==)
  10. Less Than (<), Less Than Or Equal (<=), Greater Than (>), Greater Than Or Equal (>=), in , instanceof
  11. Bitwise Left Shift (<<), Bitwise Right Shift (>>), Bitwise Unsigned Right Shift (>>>)
  12. Addition, Subtraction
  13. Multiplication(*), Division (/), Remainder(%)
  14. Exponentiation(**)
  15. Logical NOT(!), Bitwise NOT (~), Unary plus (+), unary negation (-), prefix increment,
    prefix Decrement, typeof, volid, delete, await
  16. Postfix Increment, Postfix Decrement
  17. new (without argument list)
  18. Member Access(.), Computed Member Access(...[...]), new (with argument list), Function Call, Optional Chaining (?.)
  19. Grouping

HTML

HTML(Hypertext Markup Language) is used to define the structure of a web page and its content.

An HTML element would have a start tag, content and an end tag. For example,
<h3> hello world </h3>

Attributes can be added to start tag in the form name=value as follows:
<h3 class="head-3"> hello world </h3>

CSS

CSS(Cascading Style Sheets)is used for adding style to web content.

    CSS selectors can be divide into 5 categories:
  • Simple selectors used to select elements based on name, id and class.
  • Combinator selectors used to select elements based on relationship between them.
  • Pseudo-class selectors used to select elements based on the state
  • Pseudo-element selectors used to style part of an element
  • Attribute selectorsused to select elements based on an attribute or attribute value
